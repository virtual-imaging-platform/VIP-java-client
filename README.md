# VIP-java-client

## Description
This project allow VIP to be used directly from java code through its REST API.
It is composed of 2 separate java clients generated by swagger codegen :
- The Processing module conform to the CARMIN specification (version: 0.2.1)
- The Data module, in a draft version, non-standard

## How to use
To use this API, you have to get your own APIkey by connecting to [VIP portal](https://vip.creatis.insa-lyon.fr/)
and creating your account.

The two modules can be used independently. The processing module contains the basic funtionality (pipeline information and execution submission). The data module contains file exchange functionalities.
